FizzBuzz Challenge

  • FizzBuzz is a very simple programming task. Originally it prints the numbers 1 to 100. For multiples of three it prints the word "Fizz" instead of the number and for multiples of five it prints the word "Buzz". For any number which is both a multiple of three and five, it will print the word "FizzBuzz". I changed it up a bit by allowing you to input the number you want the program to find multiples of and the range of numbers it will use to search for the multiples.
  • You will see four total FizzBuzz programs below. Each is unique in it's own way. I wrote each program in a different way, mostly for practice but it does show that I am versatile and can adapt to any challenge. I have also included a snippet of each JavaScript code used for each program. Please enjoy!
